1.Init package json
`npm init -y`

2.Install gulp globally and locally
`npm i -g gulp`
`npm i -g -D gulp`

3.Install plugins
**SCSS convert, min, and autoprefix*
`npm i -D gulp-sass sass gulp-clean-css`

**JS min*
`npm i -D gulp-minify`

4.Add gulpfile.js

5.Create functions, watch task, and default gulp

6.Run gulp
