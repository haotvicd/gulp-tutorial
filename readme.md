1.Init package json
`npm init -y`

2.Install gulp globally and locally
`npm i -g gulp`
`npm i -g -D gulp`

3.Install plugins
**Convert PUG -> html*
`npm i -D gulp-pug`

**SCSS convert, min*
`npm i -D gulp-sass sass gulp-clean-css`

**JS min, ignore file*
`npm i -D gulp-minify`

**Create live server with browser sync*
`npm i -D browser-sync`

4.Add gulpfile.js

5.Create functions, watch task, and default gulp

6.Run gulp
